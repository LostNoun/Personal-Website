HTML 5

This is a quick referance to HTML 5 tags/elements.

Tag/Element Name Description
a Anchor An anchor tag or hyperlink
body Body Creates the body of the document and is required
br Break Line/Text break
div Divide Divides document into a new section
em Emphasize Emphasizes text
h1,h2,h3,...h6 Headings Page headings h1 is the main down to h6
head Head Contains meta tags plus information about the document and is reqiered
HTML HTML Root The Root of the document and in required
img Image Creates an image with provided link
link Link Links an external file
main Main Wraps the main content
p Paragrapgh Wraps a paragraph
style Style Wraps CSS code
table Table Creates a table
tbody Table Body Wraps table body
td Table Data Creates table cells in a row
tfoot Table Foot Wraps table foot
th Table Header Creates table header cell in row
thead Table Head Wraps table head
title Title Creates tab title and is required
tr Table Row Wraps table row
var Variable Creates a variable
video Video Creates a video with provided link
!-- -- Comment Wraps comment
